## 操作步骤

1. Clone 仓库并准备 `data/` 下 4 个 parquet（见 [使用指南.md](使用指南.md)）。
2. 打开 `notebooks/02_grid_search.ipynb`，运行路径 cell 与 memmap 准备 cell。
3. 将「阶段配置」cell 中的 `grid_space` 换成你认领任务的配置。
4. 运行 `build_experiments` 与实验循环 cell。
5. 将生成的 csv/jsonl **重命名**为 `grid_search_fast_{姓名}_{任务ID}_{时间}.csv` 放入 `results/grid_search/`。
6. 按 [成员提交与贡献者署名说明.md](成员提交与贡献者署名说明.md) 提 PR 或发给维护者。
