# 数据与模型下载（GitHub Release）

训练 parquet 与 baseline 权重体积较大，**不入 Git**。请从本仓库的 **Latest Release** 下载后放到 `data/`。

**Release 页面**：<https://github.com/kamineayaka/UserBehavior-SASRec/releases>

---

## 必下：训练数据（约 410 MB）

解压后，以下 4 个文件应位于 **`data/` 根目录**（与 `data/README.md` 同级）：

| 文件 | 约大小 | 说明 |
|------|--------|------|
| `train.parquet` | ~377 MB | 训练序列 |
| `valid.parquet` | ~9 MB | 验证目标 |
| `test.parquet` | ~9 MB | 测试目标 |
| `item2idx_mapping.parquet` | ~15 MB | 物品 ID 映射 |

Release 中可能是单独 4 个文件，或打包为 zip（例如 `sasrec_data_parquet.zip`）。

**注意**：若解压后文件在子文件夹内（如 `sasrec_data_parquet/train.parquet`），请**移动到 `data/` 根目录**，不要多留一层目录。

---

用于 [`notebooks/03_goal_check.ipynb`](notebooks/03_goal_check.ipynb) 对比 MostPop / ItemCF，**网格搜索与全量训练不依赖此文件**（从零训练）。

| 文件 | 放置路径 |
|------|----------|
| `baseline_sasrec_20260425_130830_hr10_0.8475_ndcg10_0.7736.pt` | `data/baseline/` |

参考指标 JSON 已在 Git：`data/baseline/baseline_sasrec_20260425_130830.json`（HR@10 / NDCG@10 约 **0.847 / 0.774**，全量用户）。

## 方式 A：自动下载（推荐）

需安装 [GitHub CLI](https://cli.github.com/) 并完成 `gh auth login`。

在仓库根目录（`UserBehavior-SASRec/`）执行：

```bash
python scripts/download_release_assets.py
```

覆盖已有文件：`python scripts/download_release_assets.py --force`

维护者若 Release 标签不是 Latest，可编辑脚本顶部 `RELEASE_TAG`（例如 `v1.0-data`）。

---

## 方式 B：手动下载

1. 打开 [Releases](https://github.com/kamineayaka/UserBehavior-SASRec/releases)，进入 **Latest**。
2. 在 **Assets** 中下载 parquet（或 zip）及可选的 `.pt`。
3. 将 4 个 parquet 放到 `data/`；将 `.pt` 放到 `data/baseline/`。

---

## 校验

```bash
python -c "from pathlib import Path; d=Path('data'); print([p.name for p in sorted(d.glob('*.parquet'))])"
```

应输出 4 个文件名。更多字段说明见 [data/README.md](data/README.md)。
